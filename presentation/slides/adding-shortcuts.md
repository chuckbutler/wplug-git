##  Adding Shortcuts

Let's add some functionality to our shell to use git quickly.

> $EDITOR aliases

    alias gs='git status'
    alias gl='git l'
    alias gc='git commit'
    alias gco='git checkout'
    alias gbr='git checkout -branch'

Commit these changes to the repository
    
    git add aliases
    git commit -m "Added git aliases"
